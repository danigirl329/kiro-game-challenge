# User Context - Super Kiro World

## Project Details
- **Game Type**: Platformer (Super Kiro World)
- **Technology Stack**: Vanilla JavaScript, HTML5 Canvas
- **Character Sprite**: kiro-logo.png

## Game Specifications
### Physics
- Gravity: 0.5
- Jump Power: 12
- Movement Speed: 5
- Friction: Applied to horizontal movement

### Features
- Platform-based level design
- Scrolling camera (smooth follow)
- Collectibles: gems and coins
- Score counter
- Lives system (hearts)
- Level completion and restart
- Collision detection (platforms and collectibles)

### Design Preferences
- Responsive platforming feel
- Simple and playable
- Browser-based
- Kiro brand colors from game-style-guide.md
